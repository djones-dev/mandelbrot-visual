# Fullstack Mandelbrot Visualizer
This is a full stack web application that will visualize the mandelbrot set through interactive controls.

## Features
- Interactive zoom and pan
- Adjustible iteration depth
- Realtime visualization updates

## Tech Stack
- **Backend:** Python, FastAPI, NumPy for computations
- **Frontend:** React, Next.js, Canvas API

## Getting Started

### Prerequisites
- Python 3.12
- Node.js 18+
- Docker (optional)
